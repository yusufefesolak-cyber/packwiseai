# packwiseai
PackWise AI: E-ticaret satıcıları için iade ve hasar riskini önleyen akıllı asistan. Gemini API gücüyle "Agentic" yapıda çalışır; ürün görselinden materyal tanıma, açıklanabilir risk skorlaması (XAI), müşteri yorumu simülasyonu ve SEO uyumlu ilan üretimi sunar. (BTK Akademi Hackathon'26)
